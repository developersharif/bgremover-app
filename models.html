How to Use These Models

1. Download the desired ONNX model.

2. Place the downloaded model file in the appropriate directory of your BG Remover installation.

3. In the BG Remover Model Settings>Chose Custom Model>yes>save, select the model you wish to use for background removal.

4. Experiment with different models to find the best one for your specific use case.

Note: The premium version of BG Remover allows for easy switching between models and provides optimized settings for each.
